Hexadecimal

Coding explained

  1. Started with the Twitter Bootstrap to provide structure for design and layout
  2. Used jQuery to help with picking up code from text input:
    1. .css to identify and set colour of the main div section
    2. .val to fetch the value of the colour from the user
    3. .text to set the values in the RGB table
  3. Standard javascript to produce the warning dialog
  4. Used @font-face CSS3 property to use the OpenDyslexic font
